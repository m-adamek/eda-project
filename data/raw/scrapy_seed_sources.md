# Curated Scrapy web sources

This seed list is intentionally selective. It focuses on public documentation
and help pages that expose how organisational systems represent identity across
name, display name, preferred/chosen name, legal name and pronoun fields.

## Source groups

- Microsoft Entra and Microsoft 365: identity profile fields, preferred versus
  legal name guidance, pronoun visibility and admin controls.
- SAP SuccessFactors: general display name, pronouns, user-data import fields
  and limitations across HR modules.
- Oracle HCM and Oracle Access Governance: person-name formats, Known As /
  preferred-name fields and identity attribute mappings.
- Slack and Google Workspace: collaboration-suite profile fields, pronoun
  visibility and user/admin control boundaries.
- Public Workday institutional guides: examples of how Workday preferred/legal
  name data propagates into directories, rosters, cards and connected systems.

## Methodological note

These pages should be treated as practice-oriented web evidence, not as
peer-reviewed literature. They are useful for showing how product documentation
and institutional implementation guides operationalise identity representation
in deployed organisational systems.
